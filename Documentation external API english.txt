 Title Consuming API:s Date 2022-10-11 Rev no 15  Page 6 (22)   1 INTRODUCTION  1.1 Purpose This document is intended to explain how to fetch information stored in the information systems used in Svenska Spel. It is possible to fetch information regarding draws, forecasts and results for products provided by Svenska Spel. In order to be able to fetch information, the consumer of the information is required to use the communication protocol Hypertext Transfer Protocol Secure (Wikipedia, HTTPS, 2018) and send a GET to the desired url for example https://api.www.svenskaspel.se/external/1/draw/stryktipset/draws?accesskey=madeUpAccessKey.  All api:s must include the parameter accesskey. Calls without the parameter will fail with error code 100005.  1.2 Scope This document is a technical specification. If you have any questions regarding agreements, copyright and so forth, please contact Andréas Johansson using phone number 010-120 38 43 or send him an email using andreas.johansson@svenskaspel.se.  1.3 Akronyms and Definitions API is an abbreviation for Application Programming Interface (Wikipedia, Application programming interface, 2017) that describes an address on the internet, also called a URL, where mainly computers can fetch structured data. JSON is an abbreviation for JavaScript Object Notation (Wikipedia, JSON, 2017). It is an open-standard file format that uses human-readable text to transmit data. A draw defined by Svenska Spel is either a betting pool (Wikipedia, Betting pool, 2017) with a fixed price plan or sports betting (Wikipedia, Sports betting, 2017). The draw has several states and a draw is confined within a period. Usually, the draw is first defined, then the draw is open and bets can be entered, then the draw is closed and finally finalized. A result is distributed when the draw is finalized and the outcome of the draw is determined. Svenska Spel uses REST (Wikipedia, REST, 2017) as a principle when the design of the api:s implemented.

 Title Consuming API:s Date 2022-10-11 Rev no 15  Page 7 (22)   2 API Versions 2.1 Version numbers Svenska Spels external API:s use a numeric versioning system in its URL:s. If breaking changes must be implemented in an API, that API will receive a new URL with an increased version number leaving the previous API version unchanged. 2.1.1 Example Let’s say the API to fetch current draws for product Matchen needs a change that breaks the API contract. To get the new features, use the new URL with an increased version number like: https://api.www.svenskaspel.se/external/2/draw/matchen/draws (notice the 2 in the URL) At the same time the previous version still functions excalty as before on the unchanged URL: https://api.www.svenskaspel.se/external/1/draw/matchen/draws 2.2 API removal If there are any reason where it is not possible to maintain an old version we will give notice in advance for you to update your software well before an old API version is removed.

The draw number is a numeric value representing a draw number, for example 1361. If no draw number is provide then the api gives information about the current open draw. 3.1 Supported products Svenska Spel supports in the api:s the following products:  • Trav & Gallop • Bomben • Challenge • Europatipset • Måltipset • Matchen • Powerplay • Sportkrysset • Stryktipset • Topptipset • Topptipset Europa • Topptipset family • Topptipset stryk • Vikinglotto • Eurojackpot (friday and Tuesday) • Lotto (wednesday and saturday)  • Keno • Joker • Fullträff  

=== PAGE 9 ===
 Title Consuming API:s Date 2022-10-11 Rev no 15  Page 9 (22)   3.2 What are Topptipset Family, Topptipset Stryk and Topptipset Europa? Topptipset is a product where the idea of the product is to predict the outcome of eight matches. These eight matches can be based upon Stryktipset or Europatipset. The api:s that concerns Topptipset and are based upon Stryktipset are described under Topptipset Stryk. An example of these api:s is https://api.www.svenskaspel.se/external/1/draw/topptipsetstryk/draws If Topptipset is based on Europatipset, then the api:s concerning this variant are described under Topptipset Europa. An example of these api:s is https://api.www.svenskaspel.se/external/1/draw/topptipseteuropa/draws. Topptipset Family is an product where current information regarding Topptipset based on both Stryktipset and Europatipset are included in the same api:s. An example of these api:s is https://api.www.svenskaspel.se/external/1/draw/topptipsetfamily/draws.   3.3 What is Sportkrysset? Sportkrysset is a product that works the same way as Joker and can be played together with Stryktipset and Måltipset.
